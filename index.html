KY Overture Explorer

          

            A browser-based tool for exploring Overture Maps datasets using DuckDB-WASM.
            Query 2.3 billion geospatial features directly from S3 — no backend required.
          

          

            Inspired by:
            
              SQL in the Browser: Querying 2.3 Billion Records with Zero Backend
            
          

          

          
Stack

          
    
            
  • DuckDB-WASM — SQL engine in the browser
    • 
            
  • Overture Maps Foundation — open geospatial data
    • 
            
  • Bootstrap 5 — UI framework
    • 
            
  • CodeMirror 5 — SQL editor
    • 
            
  • Leaflet.js — map rendering
    •
